# Local Models

Hover supports local OpenAI-compatible model servers.

## LM Studio

Default URL:

```text
http://localhost:1234/v1
```

Start the LM Studio local server, then use Hover Settings > Inference > Refresh.

## Ollama

Default URL:

```text
http://localhost:11434/v1
```

Install a model with Ollama, start the local service, then use Hover Settings > Inference > Refresh.

## Privacy

Local providers keep requests on the local machine or local network endpoint you configured. Hover still shows context transparency so users can see whether selected text, screen text, or screenshot context is included.

# Install Hover

## Official DMG

1. Download the DMG from the official paid/supporter link.
2. Open the DMG.
3. Drag Hover into Applications.
4. Launch Hover from Applications.
5. Follow the setup guide.

Unsigned early builds may show a macOS warning. Signed and notarized builds provide the cleanest install experience.

## Build From Source

1. Install Xcode.
2. Open `Hover.xcodeproj`.
3. Build and run the `Hover` scheme.
4. Grant only the permissions for features you want to use.

## First Run

Hover starts as a menu bar app. Use the setup guide to:

- Review permissions.
- Choose a provider.
- Test the model connection.
- Try `Command-Shift-X` on selected text.

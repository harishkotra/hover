# Trademark Policy

The source code is open, but the Hover name, logo, icon, website copy, and official build identity are not a general-purpose license to impersonate the project.

You may:

- Refer to the project as Hover when discussing, contributing to, or forking it.
- State that your project is based on Hover if that is true.

You may not:

- Publish a fork using the Hover name or icon in a way that suggests it is an official build.
- Sell or distribute unofficial builds as "Hover" without clear labeling.
- Use the official website copy, payment pages, or support identity for a fork.

Forks should use a distinct app name, bundle identifier, icon, and support contact.
